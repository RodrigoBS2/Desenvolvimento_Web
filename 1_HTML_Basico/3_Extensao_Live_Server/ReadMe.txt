LIVE SERVER:
 - Uma extensão do VsCode;
 - Atualiza automaticamente a página;
 - A página será aberto por um endereço:
    - Esse endereço é a versão numérica do seu computador (referência da sua prórpia máquina, o IP);
    - Voce pode colocar no endereço: o número (IP) ou digitar "localhost" + o número que o LiveServer te dar (canto
     inferior direito do VsCode). Lembre-se q isso já é feito automáticamente quando você ativa o LiveServer.
 - A atualização é feita toda vez que o código modificado é salvo.
